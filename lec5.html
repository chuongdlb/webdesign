1. What is jQuery?

jQuery is a fast, small JavaScript library that makes tasks like HTML document traversal, manipulation, event handling, animation, and AJAX much simpler with an easy-to-use API that works across many browsers.

Core idea: $(selector) finds elements, then you chain methods to act on them.
// Basic document ready pattern (wait until DOM prepared)
$(document).ready(function() {
	console.log('jQuery is ready!');
});

// Selecting all paragraphs and changing text color
$('p').css('color', 'darkslateblue');

// Event: clicking a button
$('#demoButton').on('click', function() {
	alert('Button clicked!');
});

2. Setup & Document Ready

Use $(document).ready() OR the shorthand $(function(){ ... }) to run code once the DOM is fully parsed.

// Long form
$(document).ready(function(){
    console.log('Ready (long form)');
});
// Shorthand
$(function(){
    console.log('Ready (shorthand)');
});
